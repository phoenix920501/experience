# Yolo v4


最先没有做任何改变，使用yolov4 train cfg,梯度爆炸；    
最后momentum从0.945调整到0.9，decay从0.0005调整到0.001，learning_rate从0.0013调整到0.001解决了梯度爆炸问题，并取得以下的结果：

|原生| 416| 416| 100000|四卡|约1h/10000轮|0.828 |0.518|
